# Few-shot примеры для генератора гипотез

Этот файл содержит positive/negative примеры для system prompt генератора.
Каждый пример структурирован как:

- `problem` — запрос пользователя;
- `bad_hypothesis` — пример плохого ответа с пояснением ошибок;
- `good_hypothesis` — корректный экспертный ответ.

## Пример 1. КГМК — снижение потерь Ni/Cu в хвостах

### problem

На основе данных комплексного анализа хвостов обогатительной фабрики КГМК
предложи технологические гипотезы по снижению потерь цветных металлов в хвостах.
Учти объёмы переработки, содержание элементов 28 и 29, классы крупности и
минералогический состав частиц. Сформулируй конкретные изменения в схеме
флотации/оборудовании, которые могут повысить извлечение.

### bad_hypothesis

**Заголовок:** Оптимизация флотации за счёт предварительного гравитационного
обогащения мелких классов с использованием интенсифицированных гидроциклонов.

**Почему плохо:**

1. Упоминает "минерализованное золото" — для никелевой фабрики КГМК это
   галлюцинация.
2. Предлагает гравитационное обогащение сульфидов Ni/Cu, что физически
   неэффективно из-за близкой плотности пирротита/пентландита/халькопирита.
3. Гидроциклон — аппарат классификации, а не концентрирования.
4. Объём "до 50 тыс. т/сут" нереалистичен для КГМК.
5. Цифра "снижение потерь на 15–20%" завышена без обоснования.
6. Не предложено ни одного изменения в реагентном режиме, pH или типе
   флотомашин.

### good_hypothesis

**Заголовок:** Магнитная сепарация над целевого класса с последующим
доизмельчением в отдельном цикле.

**Формулировка:**
Для снижения потерь Ni и Cu в хвостах КГМК предлагается ввести стадию магнитной
сепарации пирротиновых хвостов с извлечением целевого класса, содержащего
закрытый пентландит (Pnt) и халькопирит (Cp). Извлечённый магнитный продукт
направляется в отдельный цикл доизмельчения с контролем границы раскрытия
(целевой класс −44 мкм), после чего возвращается в контур основной флотации.
Это позволит раскрыть сростки и повысить извлечение элементов 28 и 29.

**Ожидаемый эффект:**
Увеличение извлечения Ni и Cu на 0.3–0.8 абсолютных процента за счёт
дополнительного извлечения закрытого пентландита. Точный эффект требует
лабораторных и пилотных испытаний.

**Риски:**

- Переизмельчение может увеличить шламовые потери.
- Необходима настройка магнитной индукции под изменчивый состав пульпы.

## Пример 2. НОФ вкр — оптимизация измельчения и классификации

### problem

По данным комплексного анализа хвостов НОФ (вкраплённая руда) предложи
технологические гипотезы по снижению потерь цветных металлов. Учти
гранулометрический состав руды, работу мельниц 3,3 м и 6 м, классификаторов
и флотационных машин.

### bad_hypothesis

**Заголовок:** Введение гравитационного доизвлечения хвостов на центробежных
концентраторах.

**Почему плохо:**

1. Гравитация не работает для сульфидов Ni/Cu из-за близкой плотности минералов.
2. Не отвечает на вопрос об оптимизации измельчения и классификации.
3. Не учитывает специфику вкраплённой руды НОФ.

### good_hypothesis

**Заголовок:** Автоматическое регулирование зазора щели конусных дробилок и
контроль гранулометрии руды перед мельницами 3,3 м и 6 м.

**Формулировка:**
Для снижения потерь Ni/Cu на НОФ предлагается внедрить систему автоматического
регулирования зазора щели конусных дробилок по сигналу датчиков гранулометрии
питающей руды. Цель — стабилизировать крупность питания мельниц 3,3 м и 6 м на
уровне, оптимальном для раскрытия сульфидов без переизмельчения. Дополнительно
рекомендуется автоматизировать подачу воды в шаровые мельницы второго этапа
измельчения для поддержания целевой плотности пульпы.

**Ожидаемый эффект:**
Снижение дисперсии гранулометрического состава питания флотации на 10–15%,
что может дать прирост извлечения Ni/Cu на 0.3–0.6 абсолютных процента.

**Риски:**

- Стоимость датчиков и системы автоматизации.
- Необходимость калибровки под изменчивость рудной массы.

## Пример 3. НОФ мед — реагентный режим и флотация

### problem

На основе анализа хвостов НОФ (медистая руда) предложи гипотезы по повышению
извлечения меди. Учти данные по плотности пульпы, времени агитации и
используемым реагентам.

### bad_hypothesis

**Заголовок:** Применение цианирования для доизвлечения золота из хвостов.

**Почему плохо:**

1. Цианирование и золото не относятся к медистой руде НОФ.
2. Медь в хвостах присутствует в сульфидных минералах, которые не извлекаются
   цианированием.
3. Решение не касается флотации, реагентов и плотности пульпы.

### good_hypothesis

**Заголовок:** Добавление реагента Finfix 300 в контактные чаны перед основной
флотацией и увеличение времени агитации.

**Формулировка:**
Для повышения извлечения меди на НОФ предлагается ввести стадию интенсивной
агитации пульпы с добавлением реагента Finfix 300 в контактных чанах перед
основной флотацией. Реагент способствует диспергированию шламов и стабилизации
пенной фазы, что улучшает флотационные свойства медистых сульфидов.
Дополнительно рекомендуется повысить целевую плотность пульпы на входе в
основную флотацию для увеличения концентрации твёрдого в камерах.

**Ожидаемый эффект:**
Увеличение извлечения Cu на 0.4–0.9 абсолютных процента при сохранении
качества медного концентрата.

**Риски:**

- Передозировка реагента может ухудшить селективность.
- Требуется пилотная отработка дозировки Finfix 300.

## Пример 4. ТОФ — классификация и возврат целевого класса

### problem

По результатам комплексного анализа хвостов ТОФ предложи технологические
гипотезы по снижению потерь цветных металлов. Учти работу классификаторов,
грохочение и возможность возврата промпродуктов.

### bad_hypothesis

**Заголовок:** Полный переход на гидроциклоны для гравитационного
концентрирования хвостов.

**Почему плохо:**

1. Гидроциклоны не предназначены для гравитационного концентрирования
   сульфидов Ni/Cu.
2. "Полный переход" требует масштабной реконструкции без обоснования.
3. Игнорируется альтернатива в виде грохочения и классификации.

### good_hypothesis

**Заголовок:** Дополнительная классификация целевого класса хвостов и возврат
промпродукта в голову процесса.

**Формулировка:**
Для снижения потерь на ТОФ предлагается организовать дополнительную
классификацию хвостов основной флотации с выделением целевого класса
(−71 +45 мкм), содержащего закрытые сульфидные частицы. Выделенный продукт
возвращается в голову процесса (измельчение или основную флотацию) для
допереработки. Альтернативно можно опробовать грохота тонкого грохочения
после второй стадии классификации для более точного разделения по крупности.

**Ожидаемый эффект:**
Снижение потерь Ni/Cu в хвостах на 0.2–0.5 абсолютных процента за счёт
возврата целевого класса на доработку.

**Риски:**

- Увеличение циркуляционной нагрузки в измельчении.
- Необходимость баланса между возвратом и пропускной способностью фабрики.

## Пример 5. Общий запрос — как не отвечать

### problem

Какие технологические изменения позволят снизить потери цветных металлов в
хвостах никелевой обогатительной фабрики?

### bad_hypothesis

**Заголовок:** Внедрение цианирования хвостов для извлечения золота.

**Почему плохо:**

1. Золото не является целевым компонентом никелевой фабрики.
2. Цианирование не извлекает сульфиды Ni/Cu.
3. Предложение не связано с флотацией и не решает поставленную задачу.

### good_hypothesis

**Заголовок:** Введение контрольной флотации хвостов основной операции с
модифицированным реагентным режимом.

**Формулировка:**
Для снижения потерь Ni/Cu предлагается ввести стадию контрольной флотации
хвостов основной операции с использованием селективных собирателей
(фосфорсодержащие эфиры или модифицированные ксантогенаты). Рекомендуется
поддерживать pH пульпы в диапазоне 9.0–9.5, усилить аэрацию в камерах
тонкого измельчения и рассмотреть применение колонных флотомашин для класса
−45 мкм. Промпродукт контрольной флотации возвращается в основной контур.

**Ожидаемый эффект:**
Прирост извлечения Ni/Cu на 0.5–1.0 абсолютных процента. Точный эффект
требует лабораторных и пилотных испытаний.

**Риски:**

- Переизмельчение может увеличить шламовые потери.
- Изменение реагентного режима может повлиять на селективность.
